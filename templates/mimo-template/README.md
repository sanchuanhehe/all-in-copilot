### What is MiMo Copilot?

MiMo Copilot is a VS Code extension that integrates Xiaomi's MiMo large language models into GitHub Copilot Chat. It allows you to use MiMo's efficient AI models as an alternative assistant directly within VS Code's native chat interface.

### Setup

1. Install this extension in VS Code
2. Get your API key from [Xiaomi MiMo Platform](https://xiaomimimo.com/)
3. Run command: `MiMo: Set API Key`
4. Enter your API key when prompted
5. Open Copilot Chat and select MiMo as the model provider
