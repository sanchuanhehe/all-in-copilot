### What is MiniMax Copilot?

MiniMax Copilot is a VS Code extension that integrates MiniMax large language models into GitHub Copilot Chat. It allows you to use MiniMax's powerful M2 series models as an alternative AI assistant directly within VS Code's native chat interface.

### Setup

1. Install this extension in VS Code
2. Get your API key from [MiniMax Open Platform](https://www.minimax.chat/)
3. Run command: `MiniMax: Set API Key`
4. Enter your API key when prompted
5. Open Copilot Chat and select MiniMax as the model provider

| Model | Description |
|-------|-------------|
| MiniMax-M2.1 | Latest and most capable |
| MiniMax-M2.1-lightning | Fast response version |
| MiniMax-M2 | Stable and reliable |
